# Vishwaa M - Portfolio

A futuristic dark-themed portfolio website showcasing my cybersecurity and full-stack development projects, experience, and technical skills.

## Features

- **Dark Futuristic UI** - Neon accents and glassmorphism design
- **Responsive Design** - Works seamlessly on desktop and mobile
- **Resume Integration** - View full resume in modal popup
- **Project Showcase** - Display of key projects with descriptions
- **Technical Skills** - Organized by categories (Security, Programming, Tools)
- **Leadership Experience** - Highlighting extracurricular involvement
- **Contact Information** - Direct links to email, GitHub, and LinkedIn

## Deployment

This portfolio is ready to be deployed on:

- **GitHub Pages**: Enable in repository settings
- **Netlify**: Drag and drop the folder
- **Vercel**: Connect GitHub repository

### Enable GitHub Pages
1. Go to Settings → Pages
2. Select "Deploy from a branch"
3. Choose `main` branch and `/root` folder
4. Your site will be available at: `https://VishwaaM04.github.io/Projects_updated/`

## Customization

### Change Neon Color
Edit the CSS variables in `assets/css/style.css`:
```css
:root {
  --neon: #6ef3ff;      /* Change this color */
  --neon-2: #a66bff;    /* Change this accent */
}
```

### Update Personal Information
Edit these fields in `index.html`:
- Name and title
- Contact details
- Resume content in modal
- Social media links

### Add New Projects
Duplicate the project card in the Projects section and update:
- Project title
- Technologies used
- Description
- Links
